# awesome-security-data-feeds
Curated list of Security Data Feeds of vendors, programming languages ecosystem, etc

## General vulnerability databases
- NVD (NIST National Vulnerability Database) - https://nvd.nist.gov/
- OSV (Open Source Vulnerabilities) - https://osv.dev/
- GitHub Advisory Database - https://github.com/advisories
- GitHub Advisory Database (data repo) - https://github.com/github/advisory-database
- Sonatype OSS Index - https://ossindex.sonatype.org/
- CISA Known Exploited Vulnerabilities (KEV) - https://www.cisa.gov/known-exploited-vulnerabilities-catalog

## Language ecosystems
- Go Vulnerability Database - https://vuln.go.dev/
- RustSec Advisory Database - https://github.com/RustSec/advisory-db
- RubySec Advisory Database - https://github.com/rubysec/ruby-advisory-db
- FriendsOfPHP Security Advisories - https://github.com/FriendsOfPHP/security-advisories
- PyPA Advisory Database - https://github.com/pypa/advisory-database
- PyUp Safety DB - https://github.com/pyupio/safety-db
- Haskell Security Advisories - https://github.com/haskell/security-advisories
- Node.js Security WG - https://github.com/nodejs/security-wg

## Vendor advisories
- Microsoft MSRC Update Guide - https://msrc.microsoft.com/update-guide
- Apple Security Updates - https://support.apple.com/en-us/HT201222
- Adobe Security Bulletins - https://helpx.adobe.com/security.html
- Android Security Bulletins - https://source.android.com/docs/security/bulletin
- Cisco Security Advisories - https://tools.cisco.com/security/center/publicationListing.x

## OS and distro feeds
- Red Hat Security Updates - https://access.redhat.com/security/security-updates/
- Ubuntu Security Notices (USN) - https://ubuntu.com/security/notices
- Debian Security Tracker - https://security-tracker.debian.org/tracker/
- Arch Linux Security Tracker - https://security.archlinux.org/
- Alpine SecDB - https://secdb.alpinelinux.org/
- SUSE Security Updates - https://www.suse.com/support/update/
- FreeBSD Security Advisories - https://www.freebsd.org/security/advisories/
- OpenBSD Security Advisories - https://www.openbsd.org/security.html
